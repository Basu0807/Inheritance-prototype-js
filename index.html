Q-2 Write code to explain prototype chaining

function Parent(name) {
        this.name = name;
      }
      
      
      Parent.prototype.greet = function() {
        console.log('Hello, my name is ' + this.name);
      };
      
      
      function Child(name, age) {
        Parent.call(this, name); // Inherit properties from Parent
        this.age = age;
      }
      
      
      Child.prototype = Object.create(Parent.prototype);
      Child.prototype.constructor = Child;
      
      
      Child.prototype.introduce = function() {
        console.log('I am ' + this.name + ' and I am ' + this.age + ' years old.');
      };
      
    
      var parent = new Parent('John');
      var child = new Child('Alice', 7);
      
     
      parent.greet();
      child.greet(); 
      child.introduce();
Output:
        Hello, my name is John
        Hello, my name is Alice
        I am Alice and I am 7 years old.

we have a Parent object constructor that takes a name parameter and sets the name property for the created object. It defines a greet method on its prototype that logs a greeting message. We also have a Child object constructor that takes name and age parameters. It calls the Parent constructor using Parent.call(this, name) to inherit the name property. It sets up the prototype chain by creating a new object using Object.create(Parent.prototype), and assigns it to the Child prototype. The constructor property is then set correctly. The Child constructor adds its own introduce method to its prototype, which logs an introduction message including the name and age. We create instances of the Parent and Child objects using new Parent('John') and new Child('Alice', 7) respectively. Through prototype chaining, both parent and child objects have access to the greet method defined in the Parent prototype. Additionally, the child object has access to its own introduce method defined in the Child prototype. When we invoke parent.greet(), it logs "Hello, my name is John". Similarly, child.greet() logs "Hello, my name is Alice". Finally, child.introduce() logs "I am Alice and I am 7 years old."

Q-3 Add a method to calculate sum of all elements in Array in array's protype, use that method to calculate sum for multiple arrays 

Array.prototype.sum = function() {
        return this.reduce(function(acc, val) {
          return acc + val;
        }, 0);
      };
      
      var arr1 = [1, 2, 3, 4, 5];
      var arr2 = [10, 20, 30, 40, 50];
      var arr3 = [3, 7, 12, 15, 21];
    
      var sum1 = arr1.sum();
      var sum2 = arr2.sum();
      var sum3 = arr3.sum();
      
      console.log("Sum of arr1:", sum1); 
      console.log("Sum of arr2:", sum2); 
      console.log("Sum of arr3:", sum3); 
 Output: Sum of arr1: 15
            Output: Sum of arr2: 150
            Output: Sum of arr3: 58

In this example, we add a sum method to the Array prototype using Array.prototype.sum. The sum method utilizes the reduce method to calculate the sum of all elements in the array. We then define three example arrays: arr1, arr2, and arr3. To calculate the sum of each array, we use the sum method directly on the array objects (arr1.sum(), arr2.sum(), arr3.sum()). The sum method calculates the sum for each array using the reduce method internally. Finally, we print the calculated sums using console.log().

Q-4 Write a JavaScript function to retrieve all the names of object's own and inherited properties.

function getAllPropertyNames(obj) {
        var propertyNames = [];
      
        var ownProperties = Object.getOwnPropertyNames(obj);
        propertyNames = propertyNames.concat(ownProperties);
      
        var prototype = Object.getPrototypeOf(obj);
      
        if (prototype !== null) {
          var inheritedProperties = getAllPropertyNames(prototype);
          propertyNames = propertyNames.concat(inheritedProperties);
        }
      
        return propertyNames;
      }
   
      var person = {
        name: 'John',
        age: 30,
      };
    
      Object.prototype.country = 'USA';
     
      var allPropertyNames = getAllPropertyNames(person);
     
      console.log(allPropertyNames);
Output:
        [ 'name', 'age', 'country' ]
